# Shell



### 常用命令

#### 通用

| 命令 | 含义               |
| ---- | ------------------ |
| pwd  | 显示当前文件夹路径 |
| rm   | 删除文件           |

#### vim

**可视模式**

| 命令 | 含义     |
| ---- | -------- |
| v    | 选择字符 |
|      |          |

**命令模式**

| 命令                       | 含义                                                         |
| -------------------------- | ------------------------------------------------------------ |
| u                          | 撤销                                                         |
| q！                        | 强制退出                                                     |
| %s/original/replacement    | 替换                                                         |
| %s/original/replacement/g  | 替换所有                                                     |
| %s/original/replacement/gc | 检索出所有的 “original” 字符串，但在替换成 “replacement” 前，先询问是否替换 |
| /search                    | 查找                                                         |





#### autojump

| 命令                               | 含义                                       |
| ---------------------------------- | ------------------------------------------ |
| j -a C /Users/float/Desktop/Coding | 添加/Users/float/Desktop/Coding为快捷目录C |
| j C                                | 跳转到快捷目录C                            |
