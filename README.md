# Blog
我的个人技术博客



### 介绍

一个基于hexo模版的博客, 部署在github.io, 预览地址 [zexiplus.github.io](zexiplus.github.io)



### 快速启动



**下载 hexo cli**

```shell
npm install -g hexo-cli
```

**新建项目**

```shell
hexo init blog
```

**启动服务**

```shell
hexo server
```



### 更改内容



**新增博客**

把markdown文件复制到 source/_posts文件夹下



**变更模版主题**

* 下载主题至 themes
* 更改 `_config.yml`文件theme字段为指定主题
